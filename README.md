# 🛍️ Fake Store API - Online Shopping Project

A simple e-commerce web application that uses the Fake Store API to display products, categories, and product details. This project demonstrates how to fetch and display data from a REST API using JavaScript.

## 🚀 Features

- Display all products
- View product details
- Filter products by category
- Responsive design
- Dynamic product cards
- Loading and error handling

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap5
- Jquery
- JavaScript (ES6)
- Fake Store API


## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/sagar-chand/fake-store-api.git
```

2. Open the project folder.

3. Open `index.html` in your browser.

No additional installation is required.

## 🌐 API Used

**Fake Store API**

Base URL:

```
https://fakestoreapi.com/
```

Example Endpoint:

```
https://fakestoreapi.com/products
```

## 📖 How It Works

1. The application sends a request to the Fake Store API.
2. Product data is fetched in JSON format.
3. Products are displayed as responsive cards.
4. Users can browse products and view details.
